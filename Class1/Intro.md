# Python 入门（进阶豪华版）

本节课将使用一个半小时的时间，带领大家领略 Python 这门语言的魅力。所介绍的内容无需掌握，大家只需对其中的内容培养出敏感性即可。

推荐B站课程：星光的Python教程

## 准备工作

1. Python 环境安装： https://www.python.org
2. IDE 安装，以 VS Code 为例： https://code.visualstudio.com
3. VS Code Python 扩展安装、Chinese 语言包扩展安装

> 常见问题汇总：
> - 在浏览器搜索时点进盗版网站，下载到病毒软件或付费软件。
> - Python 环境变量配置有问题，导致命令行无法使用 python 命令（报错：不是内部或外部命令，也不是可运行的程序）。
> - Python 环境变量被覆盖，导致命令行的 python 命令会跳转到微软商店。

遇到问题时，建议先自行搜索解决方案，或询问 AI。如需询问他人，请学会描述问题和截图（不是拍屏）。

## 知识框架

以下列出了 Python 语言的主要知识点：

1. 输入输出
2. 变量与数据类型
3. 运算符与表达式
4. 条件分支
5. 循环
6. 函数
7. 面向对象
8. 模块与包
9. 文件操作
10. 错误与异常处理
11. 类型注解

针对不同的应用场景，可能需要额外知识点：

1. 图像处理（例如使用 pillow 或 opencv）
2. 数据分析（例如分析 csv 文件或绘制图表）
3. 网络编程（例如使用 requests 库进行网页数据抓取）
4. 机器学习（例如使用 scikit-learn 或 pytorch）
5. GUI 编程（例如使用 tkinter 或 PyQt）
